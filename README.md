## Автоматический инсталлер node_exporter с авторизацией по mTSL и добавление в prometheus, с авто обновлением сертификатов и отчетами в телеграмм

**Требования**
- Перед установкой заполнить 'cfg/secrets.env' для отчетов в Telegram
- Если хост ноды удаленный: обязательно наличие домена у ноды, ключи для подключения в '/root/.ssh/node_domain_address.key' на хосте где запускаем и 'sudo' без пароля у пользователя ноды

**Установка**
- Запуск на хосте с установленным prometheus
- Предпологается работа от 'root' или открытая сессия через 'sudo -i'
- Запустить 'cert_install.sh' для генерации сертификатов, затем сам запустится 'node_exporter_install.sh' для установки 'node_exporter'

**Мелочи жизни**
- При первой установке генерируются 'root' сертификаты, если есть старые а вы выбрали что это первая установка, скрипт не будет ничего делать и выйдет с ошибкой

**Отчеты в телеграмм**
- о результатах автообновления со списком хостов где были обновлены сертификаты, где они отсутствовали, где они были просрочены, где были ошибки от отсутствующих ключей 'ssh', где было неудачное подключение по 'ssh', где была неудача проверки сроков действия сертификата.
