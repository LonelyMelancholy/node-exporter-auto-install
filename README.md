## Автоматический инсталлер node_exporter с авторизацией по mTSL и добавление в prometheus, с авто обновлением сертификатов и отчетами в телеграмм

- Запуск на хосте с prometheus
- Перед установкой заполнить secrets.env для отчетов в Telegram
- если хост ноды удаленный, обязательны ключи в /root/.ssh/node_address.key и sudo без пароля у юзера
- Запустить cert_install.sh для генерации сертификатов, затем сам запустится node_install для установки node_exporter
